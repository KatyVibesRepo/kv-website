# Roadmap

## Phase 1 — Functional MVP

- Public event pages
- General Admission and table reservations
- Stripe Checkout
- Stripe webhook fulfillment
- Reservation creation
- Basic inventory tracking
- Basic admin event inventory page
- Basic reservation check-in

## Phase 2 — Launch-ready operations

- Admin authentication
- Create/edit events from dashboard — started in v2
- Create/edit ticket types from dashboard — started in v2
- Real confirmation emails
- QR code image generation in email
- QR scanner page
- Refund/cancel workflow
- Better reporting
- Export CSV

## Phase 3 — Katy Vibes workflow integration

- Google Sheets export/sync
- Door staff mobile check-in mode
- Table assignment dashboard
- Reservation notes
- No-show reporting
- Customer tags/VIP history
- Promo codes
- Service fees

## Phase 4 — KV TableService integration

- Paid reservations appear on the host tablet
- Table assignment syncs to floor map
- Reservation card appears when the table is selected
- Moved reservations keep their reservation card
- TableService can show reserved / checked-in / seated / left statuses


## Completed in v2-admin-events patch

- Browser-based admin event creation.
- Browser-based event editing.
- Ticket/table price and inventory editing.
- Inventory safety guard that prevents reducing inventory below already sold/held units.
- America/Chicago date input/output helpers.
